# Story 10: Pose library real-time search

**Epic:** epic-02-pose-library
**Status:** in progress

## Goal

As a practitioner, I want to search poses by name in real time on the Pose Library page, so I can quickly find a specific pose without scrolling through all cards.

## Acceptance Criteria

- [ ] Search input is displayed below the category filter row, right-aligned
- [ ] Typing in the search input immediately filters visible pose cards (no page reload)
- [ ] Search is case-insensitive and matches partial English names only (not Sanskrit)
- [ ] Category filter and search work together — search applies within the active category
- [ ] Changing category resets the search input
- [ ] When search yields no results, an empty state is shown with a "Clear search" option
- [ ] Search input follows the project design system (Cormorant italic, earthy palette)
