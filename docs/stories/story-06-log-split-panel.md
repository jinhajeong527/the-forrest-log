# Story 06: Log CRUD with master-detail split panel

**Epic:** epic-03-practice-log
**Status:** done
**PR:** #6

## Goal

As a practitioner, I want to create, view, edit, and delete practice log entries without leaving the page, so the experience feels fluid and focused.

## Acceptance Criteria

- [x] Right panel switches between list / create / view / edit modes via state (no navigation)
- [x] "+ New Log" pre-fills date from selected calendar date or today
- [x] Peak pose selectable via pose picker dialog (ForrestDialog)
- [x] Condition before/after recorded as 1–5 star rating
- [x] Props toggled via icon grid (dimmed → active)
- [x] Sequence button placeholder rendered at bottom of form
- [x] Delete returns panel to list view
